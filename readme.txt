Netcraw Project Shield Cache Purge is a WordPress plugin that allows site administrators to clear the cache of web pages saved in Google Project Shield.

== Description ==

Netcraw Project Shield Cache Purge is a WordPress plugin developed by Netcraw Creative IT (www.netcraw.com) that allows site administrators to clear the cache of web pages saved in Google Project Shield.

To use the plugin, simply enter the specific site settings (API, site ID and host) and select the link or path you want to clear from the cache.

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory or install it directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Enter the specific site settings (API, site ID and host) on the plugin settings page.
4. Select the link or path you want to clear from the cache on the plugin settings page.

== Frequently Asked Questions ==

= How do I enter my site's specific settings? =

You can enter your site's specific settings (API, site ID and host) on the plugin settings page.

= How do I select the link or path I want to clear from the cache? =

You can select the link or path you want to clear from the cache on the plugin settings page.

== Configuration ==

To configure the plugin:
Enter your API key, site ID and site host (www.yoursite.com) in the "config" menu.

To clear the cache:
Go to the "clear cache" menu and select the path you want to clear from the cache or leave the path as "/" to clear the entire site's cache.
